Conoce Decred

Decred es una moneda digital autónoma y descentralizada, dirigida por su comunidad

Iníciate

Decred en Español es tu guía para el ecosistema de DCR.
La tecnología siempre está cambiando. Te recomendamos
empezar por uno de estos temas:

Elige una billetera Recomendamos usar billeteras de código abierto. Decred financia el desarrollo de billeteras para toda clase de equipos. Elegí la tuya.
Obtiene DCR El primer paso para participar de la comunidad es obtener Decred. Consulta el listado de exchanges que soportan DCR.
Participa del Staking Decred cuenta con un sistema de Proof-of-Stake. Para participar debes congelar DCR a cambio de tickets. Conoce más aquí.
Conoce DCRDEX Decred desarrolló una casa de cambio de criptomonedas descentralizada. Puedes cambiar DCR por BTC sin ceder el control de tus fondos.
Usa la Lightning Network La red Lightning es una solución de segunda capa para facilitar pagos pequeños y microtransacciones. Accede a la guía completa.
Aprende sobre Privacidad DCR es una criptomoneda pseudonima, pero la blockchain es completamente transparente. Aprende cómo usar las soluciones de privacidad de Decred.

Billeteras

Recomendamos usar billeteras de código abierto. Decred financia el desarrollo de billeteras para toda clase de equipos. Elegí la tuya.

recomendada
Decrediton

La cartera de escritorio Decrediton tiene todo lo necesario para gestionar tu DCR, usar la Lightning Network, acceder al sistema de Staking y participar de la DAO.

mobile
Android/iOS

Decred desarrolla billeteras para dispositivos móviles con sistema operativo Android o iOS. Estas no cuentan con soporte para staking.

Avanzada
Línea de comandos

La billetera de línea de comandos de Decred se mantiene actualizada con las últimas mejoras. Se recomienda que solo usuarios avanzados o desarrolladores la usen.

Obtiene Decred

Las casas de cambio de monedas digitales permiten comprar e intercambiar Decred (DCR) fácilmente. Recuerda siempre hacer tu propia investigación antes de usar un servicio. La forma más segura de guardar Decred es en una billetera donde tu controles las claves privadas.

DCRDEX

Decred desarrolló una casa de cambio de criptomonedas descentralizada. Puedes cambiar DCR por BTC sin ceder el control de tus fondos.

  • Mantienes el control de tú dinero.

  • Protegemos tú privacidad.

  • Sin comisiones de trading.

  • Completamente P2P.

  • Abierto para todos.
